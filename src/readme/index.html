Add New Page

  • Create a folder inside src.

    As example, create a folder called profile.

    Example:

              src
           |-- profile

  • Add files html, scss and js files inside the folder you created.

    Example:

              src
           |-- profile
                |-- index.html
                |-- index.scss
                |-- index.js

  • Now add a new entry point in webpack.common.js.

    Example:

              ...
          entry: {
            main: path.resolve(__dirname, "src/"),
            readme: path.resolve(__dirname, "src/readme/"),
            // more entry points here
            profile: path.resolve(__dirname, "src/profile/"),
          },
          ...

  • Now add new HtmlWebpackPlugin({...}) inside plugins in each webpack.dev.js and
    webpack.prod.js

    Example: webpack.dev.js

              ...
          plugins: [
            new HtmlWebpackPlugin({
              filename: "index.html",
              template: path.resolve(__dirname, "src/index.html"),
              chunks: [ "main" ]
            }),
            new HtmlWebpackPlugin({
              filename: "readme/index.html",
              template: path.resolve(__dirname, "src/readme/index.html"),
              chunks: [ "readme" ]
            })
            // new HtmlWebpackPlugin for each html file.
            new HtmlWebpackPlugin({
              filename: "profile/index.html",
              template: path.resolve(__dirname, "src/profile/index.html"),
              chunks: [ "profile" ]
            })
          ],
          ...

    Example: webpack.prod.js

              ...
          plugins: [
            ...
            new HtmlWebpackPlugin({
              filename: "profile/index.html",
              template: path.resolve(__dirname, "src/profile/index.html"),
              hash: true,
              chunks: [ "profile" ],
              minify: {
                removeComments: true,
                collapseWhitespace: true,
                removeAttributeQuotes: true
              }
            })
            ...
          ],
          ...

    Note: More chunks can be added in chunks array.
    Each chunk denotes an entry point.
    Chunk name must be same as entry point.
